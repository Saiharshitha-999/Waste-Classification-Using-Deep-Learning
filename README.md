# Waste Classification Using Deep Learning

## Overview
This project classifies waste images into 10 categories using Deep Learning.

Two models were implemented and compared:
- Custom CNN
- ResNet18 Transfer Learning

## Results

| Model | Test Accuracy |
|--------|---------------|
| Custom CNN | 67.70% |
| ResNet18 Transfer Learning | 75.53% |

## Technologies Used

- PyTorch
- Torchvision
- NumPy
- Matplotlib
- Scikit-learn
